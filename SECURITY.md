# Política de Segurança

## Versões Suportadas

Abaixo está a lista de versões suportadas que atualmente recebem atualizações de segurança:

| Versão | Suportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reportando uma Vulnerabilidade

Por favor, **NÃO** reporte problemas de segurança em issues públicas.

Se você descobrir uma vulnerabilidade de segurança, envie um e-mail confidencial para `security@seudominio.com`.

Forneça os seguintes detalhes em seu relato:
- Descrição da vulnerabilidade e seu impacto.
- Passos para reproduzir a vulnerabilidade.
- Possíveis mitigações, se você as conhecer.

Entraremos em contato com você no prazo de 48 horas.
